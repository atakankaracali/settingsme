## 🛸 Setup & Run Locally

```bash
# 1. Clone the repository
git clone https://github.com/yourusername/settingsme.git

# 2. Navigate into the folder
cd settingsme

# 3. Install dependencies
npm install

# 4. Start the dev server
npm run dev
```
